# Coding Challenge 04 - Retail Discount Engine

## Description

This project is a JavaScript application that calculates retail discounts for products using conditional logic, loops, and switch statements. It demonstrates how different discount rules can be applied based on product categories and pricing conditions.

## Technologies Used

- JavaScript (ES6)

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Run the JavaScript file using Node.js or your preferred JavaScript environment.

## Features

- Calculates discounts for multiple products
- Uses switch statements for category-based logic
- Applies conditional discount rules
- Processes product data using loops

## What I Learned

This project strengthened my understanding of JavaScript control structures, including switch statements, loops, conditional logic, and working with arrays of objects to solve real-world business problems.
